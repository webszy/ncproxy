<p align="center" style="font-size: 66px">NCProxy</p>

> A CLI for manange npm proxy config

# install
> npm i ncproxy -g

# usage
you can see details with `ncproxy -h`
## list
show your all proxy
> ncproxy or ncproxy list

## set 
> ncproxy set [url] [-b/-h/-s]

## reset
clear your proxy config
> ncproxy reset
